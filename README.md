# CollabVM.js

## An npm module made with TypeScript, that can interact with CollabVM servers

## 1. How to use it

### a. Install the package

To install CollabVM.js to your workspace, do `npm install collabvm.js`

To update CollabVM.js, do `npm install collabvm.js@latest`

To install a specific version of CollabVM.js, do `npm install collabvm.js@the-version`

### b. Learn JavaScript/TypeScript

This is important to start off here, else you won't know where to start and how to import a package.

### c. Read the documentation

Everything is in this exact folder: <https://github.com/befaci03/CollabVM.js/tree/main/docs>
